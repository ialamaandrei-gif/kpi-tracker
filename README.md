# KPI Tracker (Vite + React)

## Run locally
1. Install Node.js 18+ from https://nodejs.org (LTS version).
2. Open a terminal in this folder.
3. Run:
   ```bash
   npm install
   npm run dev
   ```
4. Open the shown URL (usually http://localhost:5173).

## Notes
- Your app code is in `src/LiquidGlass.jsx`.
- `src/main.jsx` mounts it into the page.
